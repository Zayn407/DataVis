# 🌐 Aid Flow Visualization Project (援助流动可视化项目)

这是一个使用 **D3.js** 和模块化 JavaScript 结构构建的交互式数据可视化项目。项目的核心目标是分析和展示全球援助（Aiddata）在不同国家之间、援助目的以及时间序列上的流动模式。

通过将 HTML 结构、CSS 样式和 D3.js 逻辑分离到 `.html` 和 `.js` 文件中，项目实现了更高的可读性和维护性。

## 文件结构

本项目包含以下关键文件，按功能和作用划分：

### 数据文件

| 文件名 | 类型 | 描述 |
| :--- | :--- | :--- |
| `aiddata-countries-only.csv` | CSV | **原始数据文件。** 包含援助流动记录，包括年份、捐助国、受助国、承诺金额和援助目的等信息。所有 `.js` 文件都从这里加载数据。 |

### 视图文件 (HTML)

这些文件仅包含页面的结构 (HTML) 和样式 (CSS)，以及对相应 JavaScript 文件的引用。

| 文件名 | 描述 |
| :--- | :--- |
| `vis_all.html` | **主入口文件。** 提供统一的界面头部和导航切换器，并使用 `iframe` 容器来加载和展示其他三个可视化视图。 |
| `vis1.html` | 可视化 1 的结构页面，用于展示国家间援助网络概览和双边流向细节。 |
| `vis2.html` | 可视化 2 的结构页面，用于展示援助流向不同目的的分布情况 (如弦图和饼图)。 |
| `vis3.html` | 可视化 3 的结构页面，用于展示援助流动的时序变化模式 (如带时间滑块的动态网络图)。 |

### 逻辑文件 (JavaScript)

这些文件包含所有 D3.js 数据处理、可视化渲染和用户交互逻辑。

| 文件名 | 描述 |
| :--- | :--- |
| `vis_all.js` | 负责处理 `vis_all.html` 中的**视图切换**逻辑 (即控制左下角按钮和 `iframe` 切换)。 |
| `vis1.js` | 包含**可视化 1** (援助网络/双边条形图) 的所有 D3.js 渲染函数和交互处理逻辑。 |
| `vis2.js` | 包含**可视化 2** (目的分类弦图/饼图) 的所有 D3.js 渲染函数和交互处理逻辑。 |
| `vis3.js` | 包含**可视化 3** (时间轴/动态网络图/时序折线图) 的所有 D3.js 渲染函数和交互处理逻辑。 |

## 🚀 如何运行项目

**重要提示：** 由于浏览器安全限制（CORS策略禁止 `file://` 协议加载本地数据），您必须使用一个本地 Web 服务器运行此项目。

### 推荐方法：使用 Python 启动本地服务器

1.  打开命令行 (Terminal/CMD)，导航到包含所有 `.html`, `.js` 和 `.csv` 文件的项目文件夹。
2.  运行以下命令启动一个简易 Web 服务器（默认端口 `8000`）：

    ```bash
    # 推荐使用 Python 3
    python -m http.server 8000 
    ```

3.  在浏览器中打开以下地址来访问主界面：

    ```
    http://localhost:8000/vis_all.html
    ```

## ⚙️ 技术栈

* **数据可视化库：** D3.js (版本 7)
* **前端语言：** HTML5, CSS3, ES6+ JavaScript
* **数据格式：** CSV
